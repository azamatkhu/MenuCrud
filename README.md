# Ejercicio 7 - MenuCRUD

**Implementa un menú en consola para elegir la operación CRUD a realizar.**

En este ejercicio he realizado un Menu para manejar los operaciones CRUD (Insertar, Actualizar, Eliminar y Mostrar).
Se puede manejar atraves de consola, poniendo el numero de opcion y segun la opcion hacer la operacion.
